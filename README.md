# Project-Bezel-Lightgun-Games-
Bezels for Emulators MAME and Others
Project Bezel for Lightgun Games By El Julo

Hi ! Thanks for download the mod. I gave my best to set up enjoyable bezels for lightgun games, mostly are for MAME emulator, but i also join some Reshade presets for Emulator like duckstation and PCSX2. Hope you will enjoy them !
I joined 2 versions : the 1080p and the 4K. Select the one you want, i Don't have a 4K monitor, so if you have troubles, just ask me to fix it. 

For MAME :

Just copy the artwork folder into your MAME folder. If the bezel don't appears, press TAB -> Video Options -> Screen #0 -> and choose Bezel Artwork. It should be fine. 

For PCSX2 or Duckstation or Flycast.

Install reshade (follow tutorial) choose the render version you have on the emulator (Vulkan/OpenGL/DirectX), 

Extract the zip contains Project Bezel by El Julo and reshade-shaders folders and paste them into your emulator folder (the one contains the .exe). 
The steps for activate the presets is inside the pdf with screen captures for those who don't know how to use reshade (it's really easy).

Test with MAME 0.286 and 0.287.
PCSX2 1.7 Nightly.

Known Bugs :

- Point Blank 2 bezel on MAME doesn't show up.
